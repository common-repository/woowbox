Apply different skins for your WP galleries. Masonry Gallery, Slider, Grid Gallery, Justify Gallery...

== Description ==

Simple and lightweight plugin which allows you to add skins to your default wordpress `[gallery]` galleries.

You can also apply different skins individually to each gallery shortcode or disable applying a skin for particular shortcode.

Works with Elementor Page Builder. Perfect gallery for AJAX WordPress Themes. Compatible with all themes and SEO optimized gallery.
Doesn't break your Posts: even if you uninstall the plugin your galleries automatically reverted back to default WordPress gallery look.

= Features include: =

* Supports YouTube and Vimeo videos in all skins
* Automatically add skin to all your WordPress Classic [gallery] shortcodes
* Compatible with Elementor Page Builder
* Compatible with AJAX WordPress Themes
* Add links to your images in gallery
* Create galleries from Media Library right on the Edit Post page
* Drag n drop reordering of images
* Load scripts and styles only on pages that have galleries
* Highly customizable settings for each of the gallery skin
* Multisite support

== Installation ==

1. Install WoowBox either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. For Classic Editor click Add Media -> Create Gallery. (Apply different skin to the gallery & change skin settings if needed).
For Gutenberg editor add Clasic Editor block and then repeat steps above.
For Elementor editor search for WoowBox Gallery block.
